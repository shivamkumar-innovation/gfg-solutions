<h2><a href="https://www.geeksforgeeks.org/problems/partition-array-for-maximum-sum/1">Partition Array for Maximum Sum</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an integer array <strong>arr[] </strong>and an integer<strong> k</strong>, partition the array into <strong>contiguous subarrays</strong> such that each subarray has a length of <strong>at most</strong> k. After partitioning, replace every element in each subarray with the maximum element present in that subarray.&nbsp;</span></p>
<p><span style="font-size: 18px;">Compute the <strong>maximum</strong> possible sum of the array after performing this operation.</span></p>
<p><strong><span style="font-size: 18px;">Examples :</span></strong></p>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr = [4, 1, 2, 3], k = 2
<strong>Output: </strong>14
<strong>Explanation: </strong></span><span style="font-size: 18px;">One optimal partition is [4, 1] and [2, 3]. Therefore, the maximum possible sum will be 4 + 4 + 3 + 3 = 14.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr[] = [2, 2], k = 1
<strong>Output: </strong>4<br><strong>Explanation: </strong>The array is partitioned as [2] and [2]. Hence, the sum is 2 + 2 = 4.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input:</strong> arr[] = [4, 6, 1, 1, 1], k = 2
<strong>Output: </strong>18<br><strong>Explanation<span style="font-size: 14pt;">: </span></strong></span><span style="font-size: 14pt;">One optimal partition is [4], [6, 1] and [1, 1], Therefore, the maximum possible sum will be 4 + 6 + 6 + 1 + 1 = 18.</span></pre>
