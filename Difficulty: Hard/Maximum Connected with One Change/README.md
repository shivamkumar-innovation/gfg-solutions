<h2><a href="https://www.geeksforgeeks.org/problems/maximum-connected-group/1">Maximum Connected with One Change</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">You are given a square&nbsp;binary grid. A grid is considered binary if every value in the grid is either&nbsp;<strong>1 or 0.&nbsp;</strong></span><span style="font-size: 18px;">You can change&nbsp;at most one&nbsp;cell in the grid from&nbsp;<strong>0 to 1</strong>.&nbsp;</span><span style="font-size: 18px;">You need to find the largest group of connected&nbsp;&nbsp;<strong>1's</strong>.&nbsp;</span><span style="font-size: 18px;">Two cells are said to be connected if both are&nbsp;adjacent(top, bottom, left, right)&nbsp;to each other and both have the same value.</span></p>
<p><span style="font-size: 18px;"><strong>Examples :</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong></span><span style="font-size: 18px;">grid[][] = [[1, 1], [0, 1]]<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927487/Web/Other/blobid1_1777636161.png" width="98" height="86">
<strong>Output:</strong> 4
<strong>Explanation:</strong> By changing the only 0 to 1, all the 1s become connected, resulting in a connected group of size 4.

<pre><span style="font-size: 18px;"><strong>Input:</strong> grid[][] = [[1, 0, 1], [1, 0, 1], [1, 0, 1]]<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927487/Web/Other/blobid5_1777636343.png" height="100"></span><span style="font-size: 18px;">
<strong>Output:</strong> 7
<strong>Explanation:</strong> By changing any 0 in the middle column to 1, the two connected groups merge into a single connected group of size 7.<br><img src="https://media.geeksforgeeks.org/img-practice/prod/addEditProblem/927487/Web/Other/blobid4_1777636323.png" height="100"><br></span></pre>
